# Referência - Estrutura do Site Quinta do Ypuã

#   https://www.quintadoypua.com/

## Principais Seções

- Página inicial com apresentação da pousada
- Página "Sobre" com informações institucionais
- Página de acomodações (listagem e detalhes)
- Página de contato (formulário, telefone, e-mail, WhatsApp)
- Links para redes sociais (Facebook, Instagram)
- Botão de reserva rápida
- Cadastro para receber promoções

## Funcionalidades Observadas

- Visualização de acomodações disponíveis
- Informações detalhadas de cada acomodação
- Canal direto de contato (WhatsApp, e-mail, telefone)
- Integração com redes sociais
- Cadastro de e-mail para promoções

## Design e Paleta de Cores

### Análise Visual do Site
- **Cores principais**: Tons terrosos e naturais (verde, marrom, bege)
- **Estilo**: Clean e minimalista
- **Tipografia**: Fontes legíveis e elegantes
- **Layout**: Responsivo com foco na experiência do usuário
- **Imagens**: Fotos de alta qualidade das acomodações e ambiente

### Paleta de Cores Identificada no Site
- **Vermelho escuro**: rgb(126, 39, 38) / #7e2726 - cor principal identificada no site
- **Tons terrosos**: Verde, marrom, bege (cores complementares)

### Sugestão de Paleta para o Sistema
- **Primária**: Vermelho escuro (#7e2726) - cor principal do site de referência
- **Secundária**: Bege/creme (#f5f5dc) - elegância e sofisticação
- **Accent**: Dourado (#d4af37) - destaque para botões e elementos importantes
- **Neutros**: Cinza claro (#f8f9fa) e escuro (#343a40) para textos e fundos
